6. 웹페이지 완성!

- href="" 이용해서 목차에 링크 연결하기
- Duplicate(복제)해서 각각의 링크가 가리킬 파일들(1.html,2.html...) 만들기
- 파일들 각각에 맞게 내용들 수정하기
- 웹 호스팅(깃허브 Github) 이용해 웹 서버 만들기
  -> new repository(저장소) 클릭해서 웹서버 생성
  -> Repository name 정하고 Public이랑 Initialize this repository with a RENAME 누르기
